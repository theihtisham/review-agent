# Review Agent

AI-powered code review GitHub Action with line-by-line PR comments

## Installation

```bash
npm install @theihtisham/review-agent
```

## Usage

```typescript
import { } from '@theihtisham/review-agent';
```

## Features

- Written in TypeScript with full type definitions
- Zero external dependencies (minimal footprint)
- Event-driven architecture
- Comprehensive test suite
